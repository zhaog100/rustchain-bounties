## 📝 RustChain Concepts Referenced

This shanty references **5 real RustChain concepts**:

1. ✅ **Antiquity Multipliers** (1.0x to 2.5x)
   - Older hardware earns higher multipliers
   - Core 2 Duo: 1.8x, Pentium 4: 2.2x

2. ✅ **Hardware Fingerprinting**
   - Verification that hardware is real, not virtualized
   - Prevents spoofing and ensures authenticity

3. ✅ **G4 Macs**
   - PowerPC G4 processors from 1999-2005
   - High antiquity multiplier (2.0x+)

4. ✅ **Epoch Settlements**
   - Each epoch (~10 minutes) rewards are distributed
   - Network consensus confirms attestations

5. ✅ **vec_perm Collapse**
   - RustChain's unique opcode for hardware verification
   - Part of the fingerprinting mechanism
