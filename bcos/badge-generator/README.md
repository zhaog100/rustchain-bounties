# BCOS Badge Generator

A static HTML/JS badge generator for BCOS v2 certifications.

## Features

- Generate embeddable badges for your BCOS-certified project
- Support for 5 badge styles: flat, flat-square, for-the-badge, plastic, social
- Auto-generate certificate IDs
- Copy Markdown, HTML, or raw SVG URL with one click
- Matches rustchain.org vintage terminal aesthetic

## Deployment

Drop `index.html` at `rustchain.org/bcos/badge-generator/` — no backend needed.
